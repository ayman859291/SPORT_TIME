# وثيقة متطلبات تطبيق وقت الرياضة\n
## 1. نظرة عامة على التطبيق

### 1.1 اسم التطبيق\nوقت الرياضة

### 1.2 وصف التطبيق
تطبيق أندرويد مخصص لمتابعة وإدارة جدول التمارين الرياضية، يساعد المستخدمين على تسجيل تمارينهم اليومية ومراقبة تقدمهم بشكل منظم.

## 2. الوظائف الأساسية

### 2.1 تسجيل التمارين
- إضافة التمارين المختلفة مع تفاصيلها
- إمكانية إضافة صورة توضيحية لكل تمرين (اختياري)
- تسجيل عدد الجلسات لكل تمرين
- تسجيل الأوزان المستخدمة في كل جلسة
- حفظ تاريخ ووقت كل تمرين
\n### 2.2 استخلاص التقارير
- عرض تقارير شاملة عن التمارين المنجزة
- إحصائيات أسبوعية وشهرية
- رسوم بيانية توضح الأداء\n
### 2.3 معدلات التطور
- متابعة التقدم في الأوزان المرفوعة\n- قياس التحسن في عدد الجلسات
- مقارنة الأداء عبر فترات زمنية مختلفة
\n### 2.4 إضافة الملاحظات
- إمكانية إضافة ملاحظات نصية لكل تمرين
- تسجيل الملاحظات حول الأداء والشعور
- حفظ نصائح وتذكيرات شخصية

### 2.5 مميزات إضافية
- جدول تمارين قابل للتخصيص
- تذكيرات لمواعيد التمارين
- حساب فترات الراحة بين الجلسات\n- تتبع السعرات الحرارية المحروقة

## 3. أسلوب التصميم
\n### 3.1 نظام الألوان
- اللون الأساسي: أزرق رياضي (#2196F3) يعكس النشاط والحيوية
- اللون الثانوي: برتقالي محفز (#FF9800) للعناصر التفاعلية
- الخلفية: رمادي فاتح (#F5F5F5) لراحة العين

### 3.2 التفاصيل البصرية
- أيقونات رياضية واضحة لكل نوع تمرين
- بطاقات منفصلة لعرض التمارين بشكل منظم
- رسوم بيانية ملونة لعرض التقدم
- أزرار بارزة بحواف دائرية لسهولة الاستخدام\n
### 3.3 تخطيط الواجهة
- تخطيط بطاقات (Card Layout) لعرض التمارين
- قوائم منسدلة لتصنيف التمارين حسب المجموعات العضلية
- شاشة رئيسية تعرض ملخص اليوم والإحصائيات السريعة
